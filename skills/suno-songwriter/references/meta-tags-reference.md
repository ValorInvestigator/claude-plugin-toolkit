# Suno AI Meta Tags Complete Reference

## Structural Tags
| Tag | Purpose | Placement |
|-----|---------|-----------|
| `[Intro]` | Opening section, mood-setting | Start of lyrics |
| `[Verse]` / `[Verse 1]` / `[Verse 2]` | Main storytelling sections | After intro or chorus |
| `[Pre-Chorus]` | Tension builder before chorus | Between verse and chorus |
| `[Chorus]` | Repeated hook, emotional peak | After verse or pre-chorus |
| `[Post-Chorus]` | Energy sustainer after chorus | After chorus |
| `[Bridge]` | Contrasting section, new perspective | After second chorus typically |
| `[Outro]` | Closing wind-down | Near end |
| `[End]` | Hard stop, prevents trailing audio | Very last tag |
| `[Fade Out]` | Gradual volume decrease to silence | Very last tag |

## Bracket Theory (Nested Instructions)

Embed instructions directly into structural tags for precise control:

```
[Intro: 15 sec, bass + drums only, establish pocket]
[Verse: Bass and drums locked tight, vocal up front, minimal guitar]
[Pre-Chorus: Add guitar, energy rise, vocal build]
[Chorus: Full band, vocal soars, bass doubles vocal hook]
[Chorus: Explosive, Anthemic, High Pitch]
[Bridge: Reflective, calm before the storm]
[Bridge: Strip guitars, keep bass and drums, whispered vocal, tension]
[Outro: Return to intro melody, gradual fade]
```

## Performance Cues (Parenthetical)

Place inside or directly after lyrics. These direct the AI's vocal delivery:

| Cue | Effect |
|-----|--------|
| `(whispered)` | Quiet, intimate delivery |
| `(spoken)` / `(spoken word)` | Talking, not singing |
| `(belted)` | Powerful, projected singing |
| `(airy)` | Breathy, light delivery |
| `(growled)` | Aggressive, gritty |
| `(crooning)` | Smooth, melodic |
| `(screaming)` | Extreme vocal intensity |
| `(softly)` | Gentle, quiet |
| `(oh yeah)` | Ad-lib background vocal |
| `(come on)` | Energy ad-lib |
| `(humming)` | Wordless melody |
